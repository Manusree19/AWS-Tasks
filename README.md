# AWS-Tasks
This repository contains hands-on tasks performed in **Amazon Web Services (AWS)** to gain practical experience with core cloud services.
The tasks cover important AWS areas such as **Compute, Networking, Storage, Identity and Access Management, Security, Monitoring, and Management Tools**.

### Identity and Access
